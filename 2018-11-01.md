# 通过以下4大类去认识js原型链类
## 1.创建js对象的3种方法
### 第一种方式：字面量
<!--more-->
`var obj1 = {name: supenghui}`
### 第二种方式：通过构造函数
```
var M = function (name) {
  this.name = name;
}
var obj2 = new M('supenghui');
var obj3 = new Object({name: 'supenghui'});
```
### 3.使用Object.creat()方法创建
```
var P = {name: 'supenghui'};
var obj4 = Object.creat(P);
```
